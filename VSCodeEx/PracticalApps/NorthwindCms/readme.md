CMS 연습용 프로젝트 - Piranha.CMS
===
```dotnetcli
//피라냐 템플릿 설치
dotnet new -i Piranha.Templates
//설치된 템플릿 리스트
dotnet new "piranha cms" --list
//mvc 생성
dotnet new piranha.mvc

```
